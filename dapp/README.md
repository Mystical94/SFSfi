#### Pre-requisites

- Node version should be >=16.0.0
- npm version should be >=9.0.0

Clone the repository

move into the dapp folder

```sh
cd dapp
```

install dependencies using **yarn** or **npm**

```sh
yarn

or

npm install
```

start the development server
```sh
yarn dev

or

npm run dev
```

build with production mode
```sh
yarn build

or

npm run build
```
